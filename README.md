# Connectly

**Connectly** is a social interaction platform built with **React** and **HTML/CSS**. It allows users to create, read, update, and delete content, interact through likes and comments, and follow other users. The backend uses **Prisma** for database management.  

## Features

- User registration and login  
- Profile management and editing  
- Create, edit, and delete posts  
- Commenting and liking posts  
- Follow/unfollow other users  
- User homepage displaying posts and profile info  

## Tech Stack

- **Frontend:** React, HTML, CSS, SCSS  
- **Backend:** Node.js, Express  
- **Database:** Prisma ORM 
- **State Management:** React Context  
- **HTTP Requests:** Axios  
- **Data Fetching & Caching:** React Query
